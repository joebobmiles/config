# `@joebobmiles/commitlint-config-gitmoji`

My personal commitlint configuration for Gitmoji.

Install via:

```shell
$ npm i -D @joebobmiles/commitlint-config-gitmoji
```

Use via:

```json
{
    ...,
    "commitlint": {
        "extends": "@joebobmiles/eslint-config"
        ...
    },
    ...
}
```
