# `@joebobmiles/config-eslint`

My personal ESLint configuration, optimizing for readability and small commit
sizes. _Highly_ opinionated.

Install via:

```shell
$ npm i -D @joebobmiles/config-eslint
```

Use via:

```json
{
    ...,
    "eslintConfig": {
        "extends": [
            "@joebobmiles/config-eslint"
        ],
        ...
    },
    ...
}
```
